# Shader study

This repository contains a tiny framework that I use to learn graphics programming (OpenGL, GLSL).

Implemented shaders:
* Flat color
* Vertex colors
* Texture mapping
* Blinn-Phong shading
* Normal Mapping
* Parallax mapping
* Shadow mapping
